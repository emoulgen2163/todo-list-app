# ✅ Todo List App

A task management app for creating and organizing daily tasks.

## ✨ Features
- Add, edit, and delete tasks  
- Mark tasks as completed  
- Simple and clean user interface  
- Lightweight and efficient  

## 🛠️ Tech Stack
